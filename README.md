# The Project: EvalX

## The Concept:
**An automated platform that uses Vision-Language models (CLIP) and static code analysis to provide objective, multi-faceted scores for student projects.**

## Module 1 (PPT): 
Scores visual layout, text density, and image relevance.

## Module 2 (GitHub): 
Analyzes commit frequency, code structure, and readability.

## Round 2 Feature: 
An AI-Based Viva/Interview module using LLMs to test conceptual depth.
